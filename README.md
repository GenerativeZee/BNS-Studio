## Overview
**BNS Studio** is a premium, ultra-minimalist website designed for a contemporary architecture and interior design firm led by Bhavneesh Sharma. The design philosophy emphasizes clarity, precision, and timeless luxury. 

The aesthetic is built around a warm off-white (`#F9F9F7`) background paired with deep charcoal typography (`#1A1A1A`), utilizing deliberate whitespace to mirror the firm's architectural expertise.

## ✨ Features
* **Editorial Layout:** A Swiss-style grid system focusing on generous spacing, visual hierarchy, and beautiful typography.
* **Responsive Design:** Fluid mobile-first architecture ensuring seamless experiences across desktops, tablets, and phones.
* **Subtle Motion:** Custom CSS `IntersectionObserver` animations for elegant, native feeling fade-ins without heavy JS dependencies.
* **Premium Typography:** Combining `Outfit` for large, high-impact headings and `Inter` for refined, readable body copy.

## 🛠️ Technology Stack
* **HTML5:** Semantic markup structure.
* **CSS3:** Custom properties, native CSS grid/flexbox layouts, clamp() scaling, and keyframe animations.
* **Vanilla JavaScript:** Zero-dependency logic for scroll events, sticky navigation, and viewport intersection observing.

## 🚀 Getting Started

### Local Development
To run this project locally, simply clone the repository and launch a local HTTP server:

```bash
# Clone the repository
git clone https://github.com/GenerativeZee/BNS-Studio.git

# Navigate into the directory
cd BNS-Studio

# Start a local python server (or use VS Code Live Server)
python -m http.server 8080
```
Then navigate to `http://localhost:8080` in your browser.

### Deployment
This static website is fully optimized for zero-config deployment. You can easily drag-and-drop the directory into platforms like **Vercel**, **Netlify**, or **GitHub Pages**.
